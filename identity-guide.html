This identity guide demos the specific elements that make up the Allianz Ireland brand style and also demos some best practice layouts for AI communications. The styles are based on the code found on Pattern Lab.

Colours

A full list NDBX colours can be found on Pattern Lab.

Colouring can be applied to text .t-colour-key, background .t-bg-colour-key and border .t-border-colour-key elements as per below.

Product groups (Car, Home, Pet, Boat, Business, MyAllianz & Customers) have specfific colours that should be applied to backgrounds and text. A reference for these colours can be found here in Product Style.
