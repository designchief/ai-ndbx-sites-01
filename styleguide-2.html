AI NDBX Style Guide

This usage guide demos the core HTML markup for AI NDBX Sites and is based on the code found on Pattern Lab.
Further documentation, additional components and options are available on Pattern Lab and the HTML markup there should work as expected. Some customisations have been made so check this page 1st for reference.
